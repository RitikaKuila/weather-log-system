#  Weather Log System

##  Description

A simple Flask web application to manage daily weather logs.
Users can add, view, update, and delete weather data (temperature and humidity).


##  Tech Stack

* Python (Flask)
* SQLite
* SQLAlchemy
* HTML, CSS



## Project Structure

    ├── home.css

    ├── update.css


## How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
python app.py
```

3. Open in browser:

```
http://127.0.0.1:5000
```

##  Features

* Add weather data
* View all logs
* Update existing data
* Delete records
